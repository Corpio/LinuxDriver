数学公式 -- 需要给浏览器安装插件

使用 LaTeX 语法插入数学公式：

$$ E = mc^2

$$

  • 数学公式示例

    • 行内公式，这是一个行内公式：$E = mc^2$

    • 块级公式，这是一个块级公式：

      $$ \int_{a}^{b} f(x) \, dx $$

解释

  1. \int：这是积分符号，表示积分运算。
  2. _{a}：这是积分的下限，表示积分从a开始。下标使用 _ 符号。
  3. ^{b}：这是积分的上限，表示积分到b结束。上标使用^符号。
  4. f(x)：这是被积函数，表示对函数f(x)进行积分。
  5. ,：这是一个小的空格符号，用于在f(x)和dx之间添加一点空格，使得公式更美观。
  6. dx：这是积分变量，表示对变量x进行积分。

完整的解释： 这个表达式表示从a到b对函数f(x)进行积分。积分是数学中用于计算函数在某个区间内的累积量的运算。

假设我们要对函数f(x) = x^2从0到1进行积分，表达式可以写成：

$\int_{0}^{1} x^2 \, dx$

  • 更多示例

    • 行内公式

    • 斜率公式：$y = mx + b$

    • 二次方程：$ax^2 + bx + c = 0$

- 块级公式
- 勾股定理：

$$
a^2 + b^2 = c^2
$$

- 欧拉公式：

$$
e^{i\pi} + 1 = 0
$$

在Sublime Text中编辑Markdown文档，如果无法显示LaTeX公式，可能是由于以下原因：

  • 未启用MathJax：确保在Markdown Preview插件的设置中启用了MathJax。

    • 打开 Preferences -> Package Settings -> Markdown Preview -> Settings。

    • 在用户设置中添加以下配置（不含<pre>标记）：

      {
"enable_mathjax": true,
"js": [
  "https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.1/MathJax.js",
  "res://MarkdownPreview/js/math_config.js"
],
"markdown_extensions": {
  "pymdownx.arithmatex": {
      "generic": true
  }
}
}
  • 如果仍然不行，则安装latex Equation Preview插件试试。
